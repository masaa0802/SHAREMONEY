#シェアマネー（SHEREMONEY)

お金の悩みを共有し合うSNS。お金は誰しも抱える悩みであるが、なかなか知り合いや友人には話しずらい場合もある。将来の資産形成、税金対策、節税アイディア、ポイント活用方法などのノウハウを共有し、金融リテラシーも向上できると思い着手しました。
